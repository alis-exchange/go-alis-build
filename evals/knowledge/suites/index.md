# Suites

Authoring guides for the suite kinds and one lazy provider.

* [Integration suite](/suites/integration-suite.md) - assert behavioural contracts against a live gRPC surface.
* [Agent-eval suite](/suites/agent-eval-suite.md) - deterministic scorers, LLM-as-judge scores, and rubric dimensions.
* [ADK agent eval](/suites/adk-agent-eval.md) - lazy provider that discovers eval sets from a deployed ADK agent at runtime.
* [Load suite](/suites/load-suite.md) - traffic generation with SLO evaluation.
* [Infra observe suite](/suites/infra-observe-suite.md) - Cloud Monitoring snapshots over a settled lookback window.

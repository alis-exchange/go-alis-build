# Definition

```protobuf
message AgentEvalResults {
  repeated Case cases = 1;
  JudgeInfo     judge = 2;

  message Case {
    string   id         = 1;
    Status   status     = 2;
    Duration duration   = 3;
    string   session_id = 4;
    repeated Metric metrics = 5;

    message Metric {
      string   id        = 1;
      Status   status    = 2;
      optional double score = 3;
      double   threshold = 4;
      string   message   = 5;
      repeated RubricScore rubric = 6;

      message RubricScore {
        string id     = 1;
        Status status = 2;
        optional double score = 3;
      }
    }
  }

  message JudgeInfo {
    string model             = 1;
    string model_version     = 2;
    int64  judge_call_count  = 3;
    int64  judge_error_count = 4;
  }
}
```

# Field notes

## `Case`

- `Case.id` ‚Äî `{suite}.{case}`.
- `Case.status` ‚Äî rolled up from every `Metric`.
- `Case.session_id` ‚Äî optional. Populated when the case exercises a
  conversational agent and a session identifier is available (for
  example, via the ADK adapter).
- `Case.duration` ‚Äî wall-clock time for the case function.

## `Metric`

- `Metric.id` ‚Äî the id passed to `t.Score`, `t.Check`, etc.
- `Metric.score` ‚Äî present only when the metric was produced by
  `t.Score` (or an equivalent scoring adapter). Bare `t.Check`
  metrics have no `score` field.
- `Metric.threshold` ‚Äî carried straight from `t.Score`. Zero for
  metrics that were not produced by `Score`.
- `Metric.message` ‚Äî the rationale passed to `t.Score`, the
  `Checkf` format applied, or an autogenerated default on failure.
- `Metric.rubric` ‚Äî optional per-dimension sub-scores. Populated by
  scorers that emit multiple sub-metrics (typical for LLM judges).

## `JudgeInfo`

Populated when the framework knows the judge model. `judge_call_count`
and `judge_error_count` help detect judge unavailability without
requiring the caller to re-parse metrics.

# Headroom

Because `threshold` and `score` are both carried on the wire,
downstream dashboards can compute per-metric headroom (`score ‚àí
threshold`) without knowing which threshold was in effect at eval
time.
